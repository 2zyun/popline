Popline is a HTML5 Rich-Text-Editor toolbar.

Inspire from popclip. Compared to traditional RTE, popline will around the mouse when you select text.

It provide common used edit features out-of-the-box.

Easy to extend, Easy to customize.

Two modes supported, Edit mode and View mode.

The default theme popline used designed by @Pepsin, Thanks.
